--------------------------
To run this Project:
  // Frontend
  1. Open the terminal
  2. Go inside the /frontend directory
  3. Run `npm i`
  4. Then, run npm start
  5. Keep it running.
  // Backend
  1. Open another terminal(keep that frontend instance running)
  2. Go inside the /backend directory
  3. Run `npm i`
  4. Then, run npm start
  5. Keep it running.
