--------------------------
To run this Project:
  1. --- Frontend --->
  2. Open the terminal
  3. Go inside the `/frontend` directory
  4. Run `npm i`
  5. Then, run `npm start`
  6. Keep it running.

 
  1. --- Backend --->
  2. Open another terminal(keep that frontend instance running)
  3. Go inside the `/backend` directory
  4. Run `npm i`
  5. Then, run `npm start`
  6. Keep it running.
