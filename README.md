To run this Project locally :

  2. Open the terminal
  3. Go inside the frontend directory using `cd frontend`.
  4. Run `npm i`
  5. Then, run `npm start`
  6. Keep it running...
  7. Then open any browser and visit the `http://localhost:3000/`
